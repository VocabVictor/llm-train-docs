# llm-train-docs

This repo is the public learning hub for training-engineering interns.

It is built with Docusaurus and is meant to answer three questions fast:

1. Where should a beginner start?
2. What should they learn this week?
3. Where do docs, demo code, and interactive labs connect?

## Local Dev

```bash
npm install
npm run start
```

## Build

```bash
npm run build
```

## Deploy

This repo is wired for GitHub Pages with a GitHub Actions workflow.
